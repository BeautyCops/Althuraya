# Althuraya — نسخة HTML ثابتة (مؤرشفة)

> **تحديث:** الموقع الفعلي مبني بـ **React / TanStack Start** في `src/`.  
> نسخة الـ HTML الثابتة القديمة تُحفَظ داخل `legacy-static/` (راجع `README.md` في جذر المشروع للنشر).

## الهيكل (داخل `legacy-static/`)

| مسار | الدور |
|------|--------|
| `legacy-static/index.html` | صفحة رئيسية — الأقسام والنصوص (عدّلي هنا للمحتوى) |
| `legacy-static/css/tokens.css` | **ألوان ومسافات** — تغيير الهوية من ملف واحد |
| `legacy-static/css/main.css` | تخطيط ومكوّنات (بطاقات، شريط، تذييل) |
| `legacy-static/js/site.js` | قائمة الجوال + السنة في التذييل |

- لإضافة **صفحات** لاحقاً: انسخي `index.html` مثلاً إلى `about.html` واربطوها من `<nav>`.
- لإضافة **صور** من التصميم: مجلد `assets/images/` واستخدمي `<img src="assets/...">`.

**لا يوجد iframe لـ Canva** — الموقع مستقل ويدار بالكامل من المشروع.
