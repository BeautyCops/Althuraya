# Althuraya — موقع ثابت (بدون Canva)

## الهيكل (قابل للتوسع)

| مسار | الدور |
|------|--------|
| `index.html` | صفحة رئيسية — الأقسام والنصوص (عدّلي هنا للمحتوى) |
| `css/tokens.css` | **ألوان ومسافات** — تغيير الهوية من ملف واحد |
| `css/main.css` | تخطيط ومكوّنات (بطاقات، شريط، تذييل) |
| `js/site.js` | قائمة الجوال + السنة في التذييل |

- لإضافة **صفحات** لاحقاً: انسخي `index.html` مثلاً إلى `about.html` واربطوها من `<nav>`.
- لإضافة **صور** من التصميم: مجلد `assets/images/` واستخدمي `<img src="assets/...">`.

**لا يوجد iframe لـ Canva** — الموقع مستقل ويدار بالكامل من المشروع.
