# نسخة HTML ثابتة قديمة (غير مُستخدَمة للإنتاج)

هذا المجلد يحفظ **الصفحة الأولى القديمة** (`index.html` + `css` + `assets`) وكان إذا وُضِع `index.html` في جذر الاستضافة يظهر **الموقع القديم** بدل تطبيق React.

**الإنتاج:** لا ترفعي هذا المجلد كجذر للنطاق. اتبعي `README.md` في جذر المشروع (بناء + Cloudflare Worker).

لمعاينة محلية للنسخة القديمة: افتحي `legacy-static/index.html` في المتصفح (قد تحتاج خادمًا محليًا بسبب المسارات).
